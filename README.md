# Task Manager App

This is a simple Task Manager application built with React and Vite.

## Features
- Add tasks
- Mark tasks as completed
- Toggle completed tasks back to incomplete
- Uses React state and spread syntax for immutable updates

## Technologies Used
- React
- Vite
- JavaScript
- CSS
